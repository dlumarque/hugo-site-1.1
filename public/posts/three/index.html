Build a Simple Blog with Hugo in One Weekend

November 9, 2024

Creating a personal blog with Hugo is a fun, straightforward project that can be completed in a weekend. Follow these steps to get your blog up and running:

1. Install Hugo

First, install Hugo on your machine. You can use Homebrew on macOS:

brew install hugo

2. Create a New Hugo Site

Open your terminal and run:

hugo new site my-blog

This command sets up your blog’s directory structure with all the files you’ll need.

3. Pick a Theme

Hugo offers many themes you can easily add to your site. Visit the Hugo Themes website to find one you like. To add a theme, go to your blog’s folder and clone the theme into the themes directory:

git clone https://github.com/example/theme-name.git themes/theme-name

Next, update your config.toml file to set the theme:

theme = "theme-name"

4. Add Your First Post

Create a post with:

hugo new posts/first-post.md

This generates a Markdown file where you can add your content. Hugo automatically formats it into your blog’s layout.

5. Preview and Build Your Site

Run hugo server -D to preview your site locally. Once you’re happy with it, build the final version by running

hugo

which creates a public folder with your static files ready to deploy.

6. Deploy

Upload your public folder to a hosting provider like Netlify or GitHub Pages, and your blog is live!

With Hugo’s simplicity and speed, you’ll have a fully functional blog by the end of the weekend!
