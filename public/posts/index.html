The first option makes me sound both like I’m the product of a seasoned developer (or pair of developers), and thus naturally am a developer myself, and/or that I’ve undergone some specialized training to be where I am, à-la-“Designer Baby”. What even is a designer baby, anyway? Build a Simple Blog with Hugo in One Weekend

Creating a personal blog with Hugo is a fun, straightforward project that can be completed in a weekend. Follow these steps to get your blog up and running:

1. Install Hugo

First, install Hugo on your machine. You can use Homebrew on macOS:

brew install hugo

2. Create a New Hugo Site

Open your terminal and run:

hugo new site my-blog

This command sets up your blog’s directory structure with all the files you’ll need.

Since it’s written in Go, Hugo compiles content almost instantly, which means I can make changes and see results immediately.
